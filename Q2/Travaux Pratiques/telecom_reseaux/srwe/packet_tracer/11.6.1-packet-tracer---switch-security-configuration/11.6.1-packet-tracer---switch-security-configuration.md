# Packet Tracer – Switch Security Configuration

> **Objectif** : Exercice de synthèse sécurité switch — trunk sécurisé, ports inutilisés, port security, DHCP snooping, PortFast et BPDU Guard.

## Étape 1 : Créer un trunk sécurisé

Câbler **SW-1 G0/2 ↔ SW-2 G0/2** (Copper Straight-Through dans l'UI).

**Sur SW-1 et SW-2 (répéter sur chacun) :**

```
SW-1> enable
SW-1# configure terminal

SW-1(config)# vlan 100
SW-1(config-vlan)# name Native
SW-1(config-vlan)# exit

SW-1(config)# interface range g0/1 - 2
SW-1(config-if-range)# switchport mode trunk
SW-1(config-if-range)# switchport nonegotiate
SW-1(config-if-range)# switchport trunk native vlan 100
SW-1(config-if-range)# exit

SW-1(config)# end
```

## Étape 2 : Sécuriser les ports inutilisés

Ports utilisés sur SW-1 : F0/1, F0/2, F0/10, F0/24, G0/1, G0/2 → tous les autres = inutilisés.  
Les mettre dans VLAN 999 (BlackHole) + shutdown = double sécurité.

```
SW-1# configure terminal

SW-1(config)# vlan 999
SW-1(config-vlan)# name BlackHole
SW-1(config-vlan)# exit

SW-1(config)# interface range f0/3 - 9, f0/11 - 23
SW-1(config-if-range)# switchport mode access
SW-1(config-if-range)# switchport access vlan 999
SW-1(config-if-range)# shutdown
SW-1(config-if-range)# exit

SW-1(config)# end
```

Répéter sur **SW-2** (ports inutilisés = tout sauf F0/1, F0/10, F0/22, F0/24, G0/1, G0/2) :

```
SW-2# configure terminal

SW-2(config)# vlan 999
SW-2(config-vlan)# name BlackHole
SW-2(config-vlan)# exit

SW-2(config)# interface range f0/2 - 9, f0/11 - 21, f0/23
SW-2(config-if-range)# switchport mode access
SW-2(config-if-range)# switchport access vlan 999
SW-2(config-if-range)# shutdown
SW-2(config-if-range)# exit

SW-2(config)# end
```

## Étape 3 : Implémenter la Port Security sur SW-1

Ports actifs sur SW-1 : **F0/1, F0/2, F0/10, F0/24**.

> ⚠️ La port security nécessite que les ports soient en mode accès statique — `switchport mode access` doit être configuré avant `switchport port-security`.

```
SW-1# configure terminal

SW-1(config)# interface range f0/1 - 2, f0/10, f0/24
SW-1(config-if-range)# switchport mode access
SW-1(config-if-range)# switchport port-security
SW-1(config-if-range)# switchport port-security maximum 4
SW-1(config-if-range)# switchport port-security mac-address sticky
SW-1(config-if-range)# switchport port-security violation restrict
SW-1(config-if-range)# exit
```

### Configurer la MAC statique de PC sur F0/1

Pour F0/1 spécifiquement, la MAC du PC est configurée statiquement.  
Récupérer la MAC du PC via `show mac address-table` ou l'affichage de la PC dans Packet Tracer.

```
SW-1(config)# interface f0/1
SW-1(config-if)# switchport port-security mac-address <MAC_DU_PC>
! → remplacer <MAC_DU_PC> par la vraie adresse MAC ex: 0001.9715.6D0B
SW-1(config-if)# exit

SW-1(config)# end
```

## Étape 4 : Configurer le DHCP Snooping

DHCP Snooping protège contre les serveurs DHCP non autorisés — seuls les ports **trusted** (trunk vers le vrai serveur DHCP) peuvent envoyer des réponses DHCP.

### Sur SW-1 — ports trunk trusted + rate limit sur ports non trusted

```
SW-1# configure terminal

SW-1(config)# ip dhcp snooping
SW-1(config)# ip dhcp snooping vlan 10,20,99

SW-1(config)# interface range g0/1 - 2
SW-1(config-if-range)# ip dhcp snooping trust
SW-1(config-if-range)# exit

SW-1(config)# interface range f0/1 - 2, f0/10, f0/24
SW-1(config-if-range)# ip dhcp snooping limit rate 5
SW-1(config-if-range)# exit

SW-1(config)# end
```

### Sur SW-2 — activer DHCP snooping globalement

```
SW-2# configure terminal

SW-2(config)# ip dhcp snooping
SW-2(config)# ip dhcp snooping vlan 10,20,99

SW-2(config)# end
```

> 💡 Note : Le scoring DHCP snooping peut ne pas fonctionner correctement dans Packet Tracer.

## Étape 5 : Configurer PortFast et BPDU Guard

**PortFast** : passe directement un port d'accès en état Forwarding sans attendre la convergence STP (30–50s) — idéal pour les ports PC.  
**BPDU Guard** : désactive immédiatement (err-disabled) un port PortFast qui reçoit un BPDU — empêche un switch non autorisé de s'y connecter.

### Sur SW-1 — PortFast et BPDU Guard sur les ports actifs

```
SW-1# configure terminal

SW-1(config)# interface range f0/1 - 2, f0/10, f0/24
SW-1(config-if-range)# spanning-tree portfast
SW-1(config-if-range)# spanning-tree bpduguard enable
SW-1(config-if-range)# exit

SW-1(config)# end
```

### Sur SW-2 — PortFast par défaut sur tous les ports d'accès

`spanning-tree portfast default` active PortFast automatiquement sur tous les ports non-trunk — plus efficace que de configurer port par port.

```
SW-2# configure terminal

SW-2(config)# spanning-tree portfast default

SW-2(config)# end
```

## Résumé des commandes clés

| Commande | Rôle |
|----------|------|
| `switchport nonegotiate` | Désactive DTP sur le port trunk |
| `switchport access vlan 999` + `shutdown` | Isole les ports inutilisés dans BlackHole |
| `switchport port-security maximum 4` | Max 4 MACs par port |
| `switchport port-security mac-address sticky` | Apprend les MACs dynamiquement |
| `switchport port-security mac-address <MAC>` | Configure une MAC statique autorisée |
| `switchport port-security violation restrict` | Violation : drop + log, port reste actif |
| `ip dhcp snooping` | Active le DHCP snooping globalement |
| `ip dhcp snooping vlan <vlans>` | Active le snooping sur des VLANs spécifiques |
| `ip dhcp snooping trust` | Marque un port comme trusted (trunk) |
| `ip dhcp snooping limit rate 5` | Limite les paquets DHCP à 5/s sur ports non-trusted |
| `spanning-tree portfast` | Active PortFast sur un port d'accès |
| `spanning-tree bpduguard enable` | Active BPDU Guard sur un port |
| `spanning-tree portfast default` | Active PortFast sur tous les ports d'accès du switch |

---

> ⚠️ **Résumé des mesures de sécurité appliquées** :
> - **Trunk sécurisé** : statique + `nonegotiate` + VLAN natif non-default (100)
> - **Ports inutilisés** : shutdown + VLAN BlackHole (999)
> - **Port Security** : max 4 MACs + sticky + MAC statique sur F0/1 + violation restrict
> - **DHCP Snooping** : trunks trusted, ports accès limités à 5 paquets/s
> - **PortFast + BPDU Guard** : convergence rapide + protection contre switches non autorisés

## Script complet (toutes les commandes dans l'ordre)

```
! ══════════════════════════════════════════════
```
