# Packet Tracer – Inter-VLAN Routing Challenge

> **Objectif** : Exercice de synthèse Router-on-a-Stick — configurer de A à Z l'adressage, les VLANs, le trunk, les sous-interfaces et désactiver les ports inutilisés.

| VLAN | Nom            | Ports S1      |
|------|----------------|---------------|
| 10   | Faculty/Staff  | F0/11 – F0/17 |

## Étape 1 : Configurer R1

### Interface G0/0 (vers le réseau externe)

```
R1> enable
R1# configure terminal

R1(config)# interface g0/0
R1(config-if)# ip address 172.17.25.2 255.255.255.252
R1(config-if)# no shutdown
R1(config-if)# exit
```

### Sous-interfaces G0/1 (Router-on-a-Stick)

La sous-interface G0/1.88 correspond au VLAN natif — on y ajoute le mot-clé `native` dans la commande `encapsulation dot1Q` pour indiquer que ce trafic ne sera pas tagué sur le trunk.

```
R1(config)# interface g0/1.10
R1(config-subif)# encapsulation dot1Q 10
R1(config-subif)# ip address 172.17.10.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface g0/1.20
R1(config-subif)# encapsulation dot1Q 20
R1(config-subif)# ip address 172.17.20.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface g0/1.30
R1(config-subif)# encapsulation dot1Q 30
R1(config-subif)# ip address 172.17.30.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface g0/1.88
R1(config-subif)# encapsulation dot1Q 88 native
R1(config-subif)# ip address 172.17.88.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface g0/1.99
R1(config-subif)# encapsulation dot1Q 99
R1(config-subif)# ip address 172.17.99.1 255.255.255.0
R1(config-subif)# exit

R1(config)# interface g0/1
R1(config-if)# no shutdown
R1(config-if)# exit

R1(config)# end
R1# copy running-config startup-config
```

## Étape 2 : Configurer S1

### Créer et nommer les VLANs

```
S1> enable
S1# configure terminal

S1(config)# vlan 10
S1(config-vlan)# name Faculty/Staff
S1(config-vlan)# exit

S1(config)# vlan 20
S1(config-vlan)# name Students
S1(config-vlan)# exit

S1(config)# vlan 30
S1(config-vlan)# name Guest(Default)
S1(config-vlan)# exit

S1(config)# vlan 88
S1(config-vlan)# name Native
S1(config-vlan)# exit

S1(config)# vlan 99
S1(config-vlan)# name Management
S1(config-vlan)# exit
```

### Assigner les ports aux VLANs

```
S1(config)# interface range f0/11 - 17
S1(config-if-range)# switchport mode access
S1(config-if-range)# switchport access vlan 10
S1(config-if-range)# exit

S1(config)# interface range f0/18 - 24
S1(config-if-range)# switchport mode access
S1(config-if-range)# switchport access vlan 20
S1(config-if-range)# exit

S1(config)# interface range f0/6 - 10
S1(config-if-range)# switchport mode access
S1(config-if-range)# switchport access vlan 30
S1(config-if-range)# exit
```

### Désactiver les ports non assignés

Les ports F0/1 à F0/5 ne sont assignés à aucun VLAN — ils doivent être désactivés pour la sécurité. Idem pour G0/2 qui n'est pas utilisé.

```
S1(config)# interface range f0/1 - 5
S1(config-if-range)# shutdown
S1(config-if-range)# exit

S1(config)# interface g0/2
S1(config-if)# shutdown
S1(config-if)# exit
```

### Configurer le trunk sur G0/1

VLAN 88 est le VLAN natif — le trafic non tagué sur ce lien sera associé à VLAN 88.

```
S1(config)# interface g0/1
S1(config-if)# switchport mode trunk
S1(config-if)# switchport trunk native vlan 88
S1(config-if)# exit
```

### Configurer la SVI de gestion

```
S1(config)# interface vlan 99
S1(config-if)# ip address 172.17.99.10 255.255.255.0
S1(config-if)# no shutdown
S1(config-if)# exit

S1(config)# ip default-gateway 172.17.99.1

S1(config)# end
S1# copy running-config startup-config
```

## Résumé des commandes clés

| Commande | Rôle |
|----------|------|
| `encapsulation dot1Q <vlan> native` | Associe la subinterface au VLAN natif (trafic non tagué) |
| `interface range f0/1 - 5` | Sélectionne un groupe de ports |
| `shutdown` sur ports inutilisés | Désactive les ports non assignés (sécurité) |
| `ip default-gateway <ip>` | Passerelle du switch pour joindre d'autres réseaux |
| `switchport trunk native vlan <num>` | Définit le VLAN natif du trunk |

---

> ⚠️ **Points d'attention spécifiques à cet exercice** :
> - Le VLAN natif est **88** (pas 1 ni 99) — bien configurer `native` des deux côtés (subinterface R1 + trunk S1).
> - Les ports **F0/1–F0/5** doivent être **shutdown** — exigence de sécurité explicitement demandée.
> - Le Server est dans le réseau `172.17.50.0/24` qui n'a pas de sous-interface dédiée sur R1 — il est accessible via le réseau G0/0.

## Script complet (toutes les commandes dans l'ordre)

```
! ══════════════════════════════════════════════
! ─── Routeur R1 ───────────────────────────────
! ══════════════════════════════════════════════

    enable

        configure terminal

            ! ─── Global Config ───────────────
            interface g0/0
                ! ─── Interface Config ────────
                ip address 172.17.25.2 255.255.255.252
                no shutdown
                exit

            interface g0/1.10
                ! ─── Subinterface Config ─────
                encapsulation dot1Q 10
                ip address 172.17.10.1 255.255.255.0
                exit

            interface g0/1.20
                ! ─── Subinterface Config ─────
                encapsulation dot1Q 20
                ip address 172.17.20.1 255.255.255.0
                exit

            interface g0/1.30
                ! ─── Subinterface Config ─────
                encapsulation dot1Q 30
                ip address 172.17.30.1 255.255.255.0
                exit

            interface g0/1.88
                ! ─── Subinterface Config ─────
                encapsulation dot1Q 88 native
                ip address 172.17.88.1 255.255.255.0
                exit

            interface g0/1.99
                ! ─── Subinterface Config ─────
                encapsulation dot1Q 99
                ip address 172.17.99.1 255.255.255.0
                exit

            interface g0/1
                ! ─── Interface Config ────────
                no shutdown
                exit

            end

        copy running-config startup-config


! ══════════════════════════════════════════════
! ─── Switch S1 ────────────────────────────────
! ══════════════════════════════════════════════

    enable

        configure terminal

            ! ─── Global Config ───────────────
            vlan 10
                ! ─── VLAN Config ─────────────
                name Faculty/Staff
                exit

            vlan 20
                ! ─── VLAN Config ─────────────
                name Students
                exit

            vlan 30
                ! ─── VLAN Config ─────────────
                name Guest(Default)
                exit

            vlan 88
                ! ─── VLAN Config ─────────────
                name Native
                exit

            vlan 99
                ! ─── VLAN Config ─────────────
                name Management
                exit

            interface range f0/11 - 17
                ! ─── Interface Config ────────
                switchport mode access
                switchport access vlan 10
                exit

            interface range f0/18 - 24
                ! ─── Interface Config ────────
                switchport mode access
                switchport access vlan 20
                exit

            interface range f0/6 - 10
                ! ─── Interface Config ────────
                switchport mode access
                switchport access vlan 30
                exit

            interface range f0/1 - 5
                ! ─── Interface Config ────────
                shutdown
                ! → ports non assignés désactivés
                exit

            interface g0/2
                ! ─── Interface Config ────────
                shutdown
                ! → port Gigabit non utilisé désactivé
                exit

            interface g0/1
                ! ─── Interface Config ────────
                switchport mode trunk
                switchport trunk native vlan 88
                exit

            interface vlan 99
                ! ─── Interface Config ────────
                ip address 172.17.99.10 255.255.255.0
                no shutdown
                exit

            ip default-gateway 172.17.99.1

            end

        copy running-config startup-config


! ══════════════════════════════════════════════
! ─── PCs (configuration via GUI) ─────────────
! ══════════════════════════════════════════════
! PC1 : IP 172.17.10.21 / 255.255.255.0 / GW 172.17.10.1
! PC2 : IP 172.17.20.22 / 255.255.255.0 / GW 172.17.20.1
! PC3 : IP 172.17.30.23 / 255.255.255.0 / GW 172.17.30.1


! ══════════════════════════════════════════════
! ─── Vérification connectivité ────────────────
! ══════════════════════════════════════════════

! PC1 > ping 172.17.10.1    → ✅ (R1 gateway)
! PC1 > ping 172.17.20.22   → ✅ (PC2, inter-VLAN)
! PC1 > ping 172.17.30.23   → ✅ (PC3, inter-VLAN)
! PC1 > ping 172.17.99.10   → ✅ (S1 management)
! PC1 > ping 172.17.50.254  → ✅ (Server)
! S1  > ping 172.17.99.1    → ✅ (R1)
```
