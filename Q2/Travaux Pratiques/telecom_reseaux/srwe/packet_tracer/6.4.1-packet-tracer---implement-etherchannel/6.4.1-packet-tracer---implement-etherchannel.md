# Packet Tracer – Implement EtherChannel

> **Objectif** : Exercice de synthèse EtherChannel — construire la topologie from scratch, configurer les trunks et implémenter PAgP et LACP entre trois switches.

---

## Topologie — Port Channel Table

| Port Channel | Switches  | Ports         | Protocole                        |
|--------------|-----------|---------------|----------------------------------|
| 1            | SWA ↔ SWB | G0/1 et G0/2  | PAgP (`desirable` ↔ `desirable`) |
| 2            | SWA ↔ SWC | F0/21 et F0/22| LACP (`active` ↔ `active`)       |
| 3            | SWB ↔ SWC | F0/23 et F0/24| LACP (`active` ↔ `passive`) — SWC initie, SWB accepte |

---

## Partie 1 : Construire la topologie

### Étape 1 – Ajouter les switches

1. **Network Devices** → sous-menu **Switches** → glisser **3 switches 2960** dans la zone de travail.
2. Arranger les switches en triangle (SWA en haut, SWB en bas-gauche, SWC en bas-droite).

### Étape 2 – Nommer les switches

> ⚠️ Cliquer sur le **nom affiché sous l'icône** (pas sur l'icône elle-même). Les noms doivent correspondre **exactement** : `SWA`, `SWB`, `SWC`.

### Étape 3 – Câbler les switches

Sélectionner **Connections** (éclair orange) → câble **Copper Straight-Through** (ligne noire diagonale).

| Port Channel | De  | Port  | Vers | Port  |
|--------------|-----|-------|------|-------|
| 1            | SWA | G0/1  | SWB  | G0/1  |
| 1            | SWA | G0/2  | SWB  | G0/2  |
| 2            | SWA | F0/21 | SWC  | F0/21 |
| 2            | SWA | F0/22 | SWC  | F0/22 |
| 3            | SWB | F0/23 | SWC  | F0/23 |
| 3            | SWB | F0/24 | SWC  | F0/24 |

## Partie 2 : Configurer EtherChannel

### Pré-requis — Configurer tous les ports en trunk statique

Avant de créer les EtherChannels, tous les ports doivent être en trunk.

**Sur SWA :**

```
SWA> enable
SWA# configure terminal

SWA(config)# interface range g0/1 - 2
SWA(config-if-range)# switchport mode trunk
SWA(config-if-range)# exit

SWA(config)# interface range f0/21 - 22
SWA(config-if-range)# switchport mode trunk
SWA(config-if-range)# exit

SWA(config)# end
SWA# show interfaces trunk

### Étape 1 – Port Channel 1 : PAgP entre SWA et SWB

Les deux côtés en `desirable` — les deux initient la négociation PAgP ✅

**Sur SWA :**

```
SWA# configure terminal

SWA(config)# interface range g0/1 - 2
SWA(config-if-range)# shutdown
SWA(config-if-range)# channel-group 1 mode desirable
SWA(config-if-range)# no shutdown
SWA(config-if-range)# exit

SWA(config)# interface port-channel 1
SWA(config-if)# switchport mode trunk
SWA(config-if)# exit

SWA(config)# end
```

**Sur SWB :**

```
SWB# configure terminal

SWB(config)# interface range g0/1 - 2
SWB(config-if-range)# shutdown
SWB(config-if-range)# channel-group 1 mode desirable
SWB(config-if-range)# no shutdown
SWB(config-if-range)# exit

SWB(config)# interface port-channel 1
SWB(config-if)# switchport mode trunk
SWB(config-if)# exit

SWB(config)# end

## Résumé des commandes clés

| Commande | Rôle |
|----------|------|
| `channel-group <n> mode desirable` | PAgP — initie la négociation |
| `channel-group <n> mode active` | LACP — initie la négociation |
| `channel-group <n> mode passive` | LACP — attend la négociation |
| `interface port-channel <n>` | Configure l'interface logique du Port Channel |
| `show etherchannel summary` | Vérifie l'état et le protocole de chaque EtherChannel |

---

> ⚠️ **Rappel** : toujours **shutdown** les ports avant `channel-group`, puis `no shutdown` après.

## Script complet (toutes les commandes dans l'ordre)

```
! ══════════════════════════════════════════════
! ─── PARTIE 1 : Topologie (actions dans l'UI) ─
! ══════════════════════════════════════════════
! 1. Ajouter 3x switch 2960
! 2. Renommer : SWA / SWB / SWC
! 3. Câbler (Copper Straight-Through) :
!    SWA G0/1  ↔ SWB G0/1   (Po1 PAgP)
!    SWA G0/2  ↔ SWB G0/2   (Po1 PAgP)
!    SWA F0/21 ↔ SWC F0/21  (Po2 LACP)
!    SWA F0/22 ↔ SWC F0/22  (Po2 LACP)
!    SWB F0/23 ↔ SWC F0/23  (Po3 LACP)
!    SWB F0/24 ↔ SWC F0/24  (Po3 LACP)


! ══════════════════════════════════════════════
! ─── PARTIE 2 : Configuration EtherChannel ────
! ══════════════════════════════════════════════

! ─── Sur SWA ──────────────────────────────────

    enable

        configure terminal

            ! ─── Global Config ───────────────
            interface range g0/1 - 2
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 1 mode desirable
                no shutdown
                exit

            interface port-channel 1
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            interface range f0/21 - 22
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 2 mode active
                no shutdown
                exit

            interface port-channel 2
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            end

        show etherchannel summary


! ─── Sur SWB ──────────────────────────────────

    enable

        configure terminal

            ! ─── Global Config ───────────────
            interface range g0/1 - 2
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 1 mode desirable
                no shutdown
                exit

            interface port-channel 1
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            interface range f0/23 - 24
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 3 mode passive
                ! → SWB accepte la négociation LACP de SWC
                no shutdown
                exit

            interface port-channel 3
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            end

        show etherchannel summary


! ─── Sur SWC ──────────────────────────────────

    enable

        configure terminal

            ! ─── Global Config ───────────────
            interface range f0/21 - 22
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 2 mode active
                no shutdown
                exit

            interface port-channel 2
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            interface range f0/23 - 24
                ! ─── Interface Config ────────
                switchport mode trunk
                shutdown
                channel-group 3 mode active
                ! → SWC initie la négociation LACP vers SWB
                no shutdown
                exit

            interface port-channel 3
                ! ─── Interface Config ────────
                switchport mode trunk
                exit

            end

        show etherchannel summary
```
